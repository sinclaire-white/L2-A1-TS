Explain the difference between any, unknown, and never types in TypeScript. 
any: এইটা যে কোনো ধরনের টাইপ গ্রহণ করে, মুলত টাইপ চেক করা অফ করে দেয়, 
যার কারণে একটা ভারিয়েবলে যে কোনো ধরনের ডাটা আসাইন করা যায় হোক সেটা নাম্বার বা বুলিয়ান বা স্ট্রিং। 
এই ক্ষেত্রে ভারিয়েবলে নাম্বার ডিক্লেয়ার করেও যদি স্ট্রিং এর মেথড কল করা হয় টাইপস্ক্রিপ্ট কোনো এরর দেয় না।
unknown: এইটা অনেকটা any টাইপের মতই কাজ করে 
শুধু পার্থক্য হচ্ছে এই ক্ষেত্রে ভারিয়েবলে নাম্বার ডিক্লেয়ার করার পর নাম্বার এর মেথড ই কল করতে হয়,
স্ট্রিং এর মেথড কল করলে টাইপস্ক্রিপ্ট এরর দেয়।
never: এইটাকে আমরা any এর বিপরীত হিসেবে চিন্তা করতে পারি। এই ক্ষেত্রে কোনো টাইপের ডাটা ই আসাইন করা যায় না ।

Provide an example of using union and intersection types in TypeScript.
Union এর ক্ষেত্রে যেকোনো একটা শর্ত সত্যি হলেই গ্রহণযোগ্য হয়, যেমন

type ID = string | number;
let userId: ID;
userId = "abc123";   
userId = 123;        

এই ক্ষেত্রে কোনো সমস্যা হবেনা, দুইটার একটা হলেই কাজ করবে।

Intersection এর ক্ষেত্রে যেকোনো একটা শর্ত সত্যি হলেই গ্রহণযোগ্য হয় না, সবগুলো সত্যি হতে হয়। যেমন

type Person = { name: string } & { age: number };
const xyz: Person = {
  name: "John",
  age: 25
};

এই ক্ষেত্রে নাম আর বয়স দুইটাই থাকতে হবে যেকোনো একটা মিসিং থাকলে এরর আসবে।
